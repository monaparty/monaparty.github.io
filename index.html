ABOUT MONAPARTY

Monaparty is a crypto-asset platform based on Counterparty and stored on Monacoin blockchain.

Monaparty is an open platform. API is open and maintained by some members.

And Monaparty is an ecosystem. Powerful users provides their services using Monaparty.
